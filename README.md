
# Creación de claves con Avalonia-Windows Forms

Este es un ejemplo de la logica y los componentes necesarios para 
generar claves que contengan al menos 6 caracteres y como mucho 12, entre los cuales tengan al menos 1 mayúscula y 1 número

Se ve el funcionamiento de un notifyPropertyChanged y como la interfaz escucha los cambios que genera el usuario a traves del ViewModel
